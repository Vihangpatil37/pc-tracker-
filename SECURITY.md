# Security Policy

## Supported Versions

Currently, only the latest release of FocusOS is officially supported for security updates. 
Since FocusOS operates entirely locally without cloud accounts or networking capabilities, the attack surface is significantly reduced.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within FocusOS (e.g., local database injection, privilege escalation, or unauthorized file access), please DO NOT open a public issue.

Instead, please email the maintainer directly at **security@example.com** (Replace with your actual security email when ready).

We take security seriously and will attempt to address vulnerabilities immediately with a patched release.
